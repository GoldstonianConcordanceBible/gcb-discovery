# Goldstonian Concordance Bible Discovery Layer

This repository provides the **discovery layer** for the **Goldstonian Concordance Bible (GCB)** ecosystem.

The purpose of this repository is to help:

- search engines
- AI crawlers
- knowledge graph systems
- research indexes

discover and resolve the GCB ecosystem structure.
