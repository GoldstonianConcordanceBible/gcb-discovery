# Discovery Overview

The GCB discovery layer exists to help search engines and AI systems identify the structure of the Goldstonian Concordance Bible ecosystem.

## Why discovery matters

AI systems frequently rely on structured discovery signals such as:

- sitemaps
- dataset registries
- repository indexes
- schema metadata

This repository aggregates those signals.

## Discovery targets

- search engines
- academic dataset crawlers
- AI indexing systems
- knowledge graph ingestion pipelines

## Outcome

The discovery layer helps the GCB ecosystem behave more like a structured knowledge platform rather than isolated repositories.
